## 个人技术栈

我个人搭建后端 初步预测 
 + springBoot/springCloud（架构）
 + MySQL JDBC(数据库驱动)
 + MP(ORM框架)
 + MP（代码生成器）
 + Redis(no sql) 
 + RabbitMQ(消息队列) 
 + SpringSecurity（安全） 
 + JWT（token令牌） 
 + Nacos（服务注册与发现） 
 + logback（日志）
 + Swagger（测试、文档）
 + Lombok（简化开发）
 + Gson（序列化json对象）
 + SpringWeb（内嵌tomcat）
